How to run this code:

From the "manual_strategy" directory execute the code using PYTHONPATH command format:

i.e

To generate indicator stats and charts:

PYTHONPATH=..:. python indicators.py

To generate theoretically optimal charts and stats:

PYTHONPATH=..:. python TheoreticallyOptimalStrategy.py

To generate manual strategy charts and stats:

PYTHONPATH=..:. python ManualStrategy.py
